Minecraft-style custom HTML elements showcase!

Here's a simple <item-slot> element.

You can specify the item, tooltip and item count using attributes:

  • item: Image link
  • tooltip: Text, supports Minecraft color codes
  • count: You can put here any text you want, lol

Also there's a <item-slot-grid> element. You can set it's size using size attribute in format "WxH" where W is width and H is height. Default size is "3x3".

I.e. using <item-slot-grid size="4x3"><item-slot-grid> will generate this:

<item-slot-grid> can have <item-slot> elements as children. It will split them in groups of W elements and display those groups as rows. First W children will be in the first row, second W children - in the second row and so on.

If there are less than W*H <item-slot> elements inside, you can add a continue attribute, which makes it fill in to W*H slots.

Here's an example (use Inspect Element to see structure). The first one won't add new slots.

But this one has a continue attribute.

Finally, you can easily display a crafting recipe with this! Just add an <item-slot> element with a result attribute inside <item-slot-grid>.

Everything gets added with one JS script - ItemSlot.js. Just add it to your page and use all of the above features!
