1. <item-slot> and <item-slot-grid>

These elements let you display Minecraft item slots and inventories.

Here's a simple <item-slot> element:

Available attributes:

  • large: Makes it large
  • item: Image link
  • tooltip: Text, supports Minecraft color codes
  • count: Text, also supports Minecraft colors
  • durability: Any integer, adds a durability bar for the item

tooltip can have any HTML code inside it!

This page can help you make a tooltip attribute.

Also there's an <item-slot-grid> element. You can set it's size using size attribute in format "WxH" where W is width and H is height. Default size is "3x3".

I.e. <item-slot-grid size="4x3"><item-slot-grid> will generate this:

<item-slot-grid> can have <item-slot> elements as children. It will split them in groups of W elements and display those groups as rows. First W children will be in the first row, second W children - in the second row and so on.

If there are less than W×H <item-slot> elements inside, you can add a continue attribute, which makes it fill in to W×H slots.

Here's an example. This element structure:

<item-slot-grid size="3x2"> <item-slot item="static/preani1.gif" tooltip="preani 1"></item-slot> <item-slot item="static/preani2.gif" tooltip="preani 2"></item-slot> <item-slot></item-slot> <item-slot></item-slot> <item-slot></item-slot> </item-slot-grid>

Will produce this:

But this one has a continue attribute:

<item-slot-grid size="3x2" continue> <item-slot item="https://minecraft.wiki/images/Invicon_Stick.png" tooltip="Stick" count="2"></item-slot> <item-slot item="https://minecraft.wiki/images/Invicon_Golden_Apple.png" tooltip="Golden Apple"></item-slot> </item-slot-grid>

2. <mc-server-plate>

This element fetches Minecraft server status and displays it in the style of in-game server list.
It uses https://api.mcsrvstat.us API.

The only attributes are address and displayname. Here's an example:

<mc-server-plate address="mc.hypixel.net" displayname="&cHypixel"></mc-server-plate>

If the server didn't respond:

<mc-server-plate address="mc.example.com" displayname="Example Server"></mc-server-plate>

And if you don't specify both address and name:

<mc-server-plate></mc-server-plate>

The JS script - MCServerPlate.js.

Click to copy - <script src="https://rumyantsev168.github.io/static/js/minecraft/MCServerPlate.js"></script>

3. <player-display>

This element renders a Minecraft skin as a Mob Vote style avatar!
You can set either a skinsrc (skin file path) or username (registered minecraft account name) attribute.
Also there's displayname, that supports color codes, and you can disable skin overlay by adding nooverlay

It renders 'Steve' profile by default.

The script is PlayerDisplay.js

Click to copy - <script src="https://rumyantsev168.github.io/static/js/minecraft/PlayerDisplay.js"></script>
